# 古の迷宮 -Ancient Labyrinth-

ブラウザだけで遊べる、見下ろし型（クオータービュー）のローグライクRPG。
依存ライブラリなしの単一HTMLファイルで動作します。

**▶ プレイ: https://hacchake.github.io/ancient-labyrinth/**

## 特徴

- 5職業（戦士・魔道士・僧侶・盗賊・狩人）、各4スキル
- 地下100階＋エンドレス、5階ごとにボス（最深部の主「虚無の支配者」）
- 装備の合成・伝説の強化、店・宝箱・地形の罠（毒沼・溶岩・トゲ）
- 敵を仲間にする「手なずけ」、深層で最大7体を引き連れる隊列（回復役あり）
- WebAudio合成のBGM／SE、実績12種＋モンスター図鑑、記録の保存

## 操作

- 移動: 矢印キー / WASD、または画面タップ
- 記録（実績・図鑑）はブラウザの localStorage に保存されます

## ファイル

- `ancient-labyrinth.html` — ゲーム本体（この1ファイルで完結）
- `index.html` — 案内トップページ
